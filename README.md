#  GENVAL

## Steup Development Environment

* Note: Use Python 3.11

- Clone this git repo

    ```bash
    git clone https://github.com/SLTDigitalLab/GenVal.git
    ```

- Navigate to cloned directory

    ```bash
    cd GenVal
    ```

- Up the containers

    ```bash
    docker compose up --build
    ```

- Started developemnt Server on

    ```bash
    localhost:18000
    ```
