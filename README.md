# dp-exec

## Steup Development Environment

- Note: Use Python 3.10+

- Clone this git repo

  ```bash
  git clone https://github.com/nzx9/dp-exec.git
  ```

- Navigate to cloned directory

  ```bash
  cd dp-exec
  ```

- Up the containers

  ```bash
  docker compose up --build
  ```

- Started developemnt Server on

  ```bash
  localhost:18000
  ```
